# Logger с поддержкой часовых поясов

Модуль для настройки логирования с поддержкой часовых поясов и удобным форматом вывода.  
Можно логировать как в консоль, так и в файл.

## Особенности

- **Часовые пояса** — поддержка через `pytz`, корректное время в логах.
- **Формат логов** — `YYYY-MM-DD HH:MM:SS LEVEL [funcName] message`.
- **Гибкая настройка**:
  - Уровень логирования: `DEBUG` или `INFO`.
  - Вывод в консоль (`CONSOLE`) или в файл (автоматически создаёт папку `src/logger/logs`).

## Пример использования

```python
from logger import configure_logger  # импортируем твою функцию

# 1. Настройка логгера
logger = configure_logger(
    tz_name="Europe/Moscow",  # часовой пояс
    logger_mode="DEBUG",      # режим логирования: DEBUG или INFO
    file_name="CONSOLE"       # выводить в консоль (или можно указать имя файла, например "app.log")
)

# 2. Использование логгера
logger.debug("Это отладочное сообщение")
logger.info("Информационное сообщение")
logger.warning("Предупреждение")
logger.error("Ошибка")
```
