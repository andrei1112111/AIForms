# model — AI ассистент для заполнения опросников 
Класс использует **OpenAI API** для работы с LLM.

## Ключевые особенности

- **AI_Interface**
  - INPUT: JSON
  - OUTPUT: JSON
  - Заполняет пустые поля JSON ответами пользователя, генерирует новые вопросы по пустым полям 

## Пример использования
INPUT
```json
{
	"previous_question": "Ты еще не задавал вопросы.",
	"user_response": "", 
	"behavior": "Вежливый и услужливый", 
	"table_description": {"ФИО": "Обязательно фио полностью", "Подарок": "Да или нет (в смысле принесёт или нет)"}, 
	"current_data": {"ФИО": "", "Подарок": ""}
 }
```
OUTPUT
```json
{
  	"data": { "ФИО": "", "Подарок": ""},
  	"question": "Пожалуйста, представьтесь полностью, указав ваши фамилию, имя и отчество."
}
```
