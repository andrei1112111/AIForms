# Frontend - AIForms

Фронтенд часть приложения AIForms, предоставляющая веб-интерфейс для работы с формами и чатом с ИИ ассистентом.

## Структура папок

```
frontend/
├── README.md
└── templates/
    ├── login.html          # Страница входа
    ├── dashboard.html      # Главная страница (панель управления)
    ├── create_form.html    # Создание новой формы
    ├── edit_forms.html     # Редактирование и удаление форм
    └── chat.html           # Чат для заполнения форм с ИИ ассистентом
```

## HTML Шаблоны

### login.html
**Назначение:** Страница входа в приложение

- Простая страница входа через Google
- Кнопка "Войти" перенаправляет на `/login`
- Минималистичный дизайн

### dashboard.html
**Назначение:** Главная страница приложения (панель управления)

**Функции:**
- Приветствие пользователя (отображение email)
- Кнопка "Создать новую таблицу" - переход к созданию формы
- Кнопка "Мои таблицы" - переход к списку созданных форм
- Кнопка "Выйти" - выход из профиля
- Инструменты для тестирования (добавление тестовых данных)

**API вызовы:**
- `POST /api/update_sheet` - добавление данных в таблицу

### create_form.html
**Назначение:** Создание новой формы

**Функции:**
- Ввод названия формы
- Ввод описания (инструкции для ассистента ИИ)
- Динамическое добавление колонок/полей:
  - Название столбца
  - Описание (что нужно узнать)
  - Удаление столбца
- Редактирование уже добавленных колонок
- Кнопка "Создать таблицу" - отправка формы на сервер
- Вывод результата с ссылками на созданную таблицу и форму для заполнения

**API вызовы:**
- `POST /api/create_sheet` - создание новой формы
  - Параметры: `title`, `description`, `columns` (массив объектов с `name` и `desc`)
  - Возвращает: `url` (ссылка на Google Sheets) и `public_link` (ссылка на публичную форму)

### edit_forms.html
**Назначение:** Просмотр, редактирование и удаление созданных форм

**Функции:**
- Загрузка и отображение списка всех форм пользователя
- Для каждой формы отображается:
  - Название
  - Описание
  - Список колонок
  - Ссылка на Google Sheets (открыть в новой вкладке)
  - Ссылка для заполнения формы (публичная ссылка)
  - Кнопка удаления
- Подтверждение перед удалением
- Обработка ошибок при загрузке

**API вызовы:**
- `GET /api/list_sheets` - получить список форм пользователя
- `POST /api/delete_form/{formId}` - удалить форму

### chat.html
**Назначение:** Интерактивный чат для заполнения формы с помощью ИИ ассистента

**Функции:**
- Отображение названия формы
- Диалоговый интерфейс для общения с ИИ ассистентом
- ИИ задает вопросы о полях формы пользователю
- Пользователь вводит ответы
- Поддержка Enter для отправки ответа
- Отображение сообщения об успешном завершении
- Сохранение данных в Google Sheets

**API вызовы:**
- `POST /api/chat/{creator_user_id}/{chat_id}/send` - отправить ответ ассистенту
  - Параметры: `user_response`, `previous_question`, `current_data`
  - Возвращает: `question` (следующий вопрос или 'success'), `data` (заполненные данные)

## Использование

1. **Вход в систему** → `login.html` → перенаправление на `/login`
2. **Главная страница** → `dashboard.html` → выбор действия
3. **Создание формы** → `create_form.html` → определение полей и описания
4. **Просмотр форм** → `edit_forms.html` → управление созданными формами
5. **Заполнение формы** → `chat.html` → интерактивный диалог с ИИ ассистентом

## Стиль и UX

- Все шаблоны используют HTML5
- CSS-стили встроены прямо в файлы (внутри тегов `<style>`)
- Поддержка русского языка (UTF-8)
- Простой и интуитивный интерфейс
- Минималистичный дизайн

## Зависимости

- Flask (серверная часть, отдает эти шаблоны)
- Клиентский JavaScript для взаимодействия с API
